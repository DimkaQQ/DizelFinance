# 📱 НАСТРОЙКА iPhone SHORTCUTS ДЛЯ АВТОМАТИЧЕСКОГО ПАРСИНГА SMS ОТ БАНКОВ

## ЧТО ЭТО ДАЁТ?

Когда приходит SMS от банка (Тинькофф, Альфа, Сбер) → iPhone автоматически:
1. Читает SMS
2. Парсит сумму, место покупки, карту
3. Отправляет данные в Telegram бота
4. Бот спрашивает: "Хотите записать эту транзакцию?"

---

## НАСТРОЙКА

### ШАГ 1: УСТАНОВИТЕ ПРИЛОЖЕНИЕ "SHORTCUTS" (КОМАНДЫ)

На iPhone уже предустановлено приложение **"Shortcuts"** (Команды/Быстрые команды).

Если его нет:
1. Откройте App Store
2. Найдите "Shortcuts"
3. Установите

---

### ШАГ 2: СОЗДАЙТЕ АВТОМАТИЗАЦИЮ

#### 1. Откройте приложение "Shortcuts"

#### 2. Перейдите на вкладку "Автоматизация" (внизу)

#### 3. Нажмите "+" (создать автоматизацию)

#### 4. Выберите "Когда приходит сообщение"

#### 5. Настройте триггер:
- **Отправитель:** Введите номер банка (например: `900` для Сбера, `Tinkoff` для Тинькофф)
- **Содержит:** Оставьте пустым (или укажите ключевое слово типа "Покупка")
- **Запускать немедленно:** Включите

---

### ШАГ 3: СОЗДАЙТЕ ДЕЙСТВИЯ

После настройки триггера, добавьте следующие действия:

#### 1. **Получить последнее сообщение**
- Действие: "Get Latest Messages"
- Количество: 1

#### 2. **Извлечь текст из сообщения**
- Действие: "Get Text from Input"
- Переменная: "Latest Messages"

#### 3. **Парсинг данных через регулярные выражения**

##### A. Извлечь сумму:
```
Действие: "Match Text"
Текст: [Текст из сообщения]
Паттерн: (\d+[\s,]?\d*\.?\d*)\s*₽
```

##### B. Извлечь место покупки:
```
Действие: "Match Text"
Текст: [Текст из сообщения]
Паттерн: (?:Покупка|Оплата)\s+(.+?)(?:\s+на|\s+₽)
```

##### C. Извлечь последние 4 цифры карты:
```
Действие: "Match Text"
Текст: [Текст из сообщения]
Паттерн: Карта\s+\*(\d{4})
```

#### 4. **Отправить данные в бот через HTTP-запрос**

```
Действие: "Get Contents of URL"
URL: https://ваш-сервер.com/webhook/transaction
Method: POST
Headers:
  Content-Type: application/json
Body (JSON):
{
  "amount": [Сумма из Match Text],
  "merchant": [Место из Match Text],
  "card": "Тинькофф",
  "date": [Текущая дата],
  "user_id": ВАШ_TELEGRAM_ID
}
```

#### 5. **Уведомление об успехе**
```
Действие: "Show Notification"
Текст: "✅ Транзакция отправлена в бот"
```

## ПРИМЕР ГОТОВОЙ АВТОМАТИЗАЦИИ

### ТРИГГЕР:
```
Когда приходит сообщение от "Tinkoff"
Содержащее: "Покупка"
```

### ДЕЙСТВИЯ:

1. **Get Latest Messages** (количество: 1)

2. **Get Text from Input** → переменная `SMS_Text`

3. **Match Text**
   - Input: `SMS_Text`
   - Pattern: `(\d+[\s,]?\d*\.?\d*)\s*₽`
   - → переменная `Amount`

4. **Match Text**
   - Input: `SMS_Text`
   - Pattern: `Покупка\s+(.+?)\s+на`
   - → переменная `Merchant`

5. **Get Contents of URL**
   - URL: `https://ваш-домен.com/webhook/transaction`
   - Method: `POST`
   - Request Body: `JSON`
   - Body:
     ```json
     {
       "amount": "{{Amount}}",
       "merchant": "{{Merchant}}",
       "card": "Тинькофф",
       "date": "{{Current Date}}",
       "user_id": "123456789"
     }
     ```

6. **Show Notification**
   - Title: "Finance Bot"
   - Body: "✅ Транзакция отправлена"

## ПАТТЕРНЫ ДЛЯ РАЗНЫХ БАНКОВ

### ТИНЬКОФФ
```
SMS от: Tinkoff, notify
Пример SMS:
"Покупка 1500 ₽. Пятёрочка. Карта *4321. Баланс: 25000 ₽"

Паттерны:
- Сумма: (\d+[\s,]?\d*\.?\d*)\s*₽
- Место: Покупка\s+\d+\s*₽\.\s*(.+?)\.\s*Карта
- Карта: Карта\s+\*(\d{4})
```

### АЛЬФА-БАНК
```
SMS от: Alfa-Bank
Пример SMS:
"Оплата 2500.00 RUB. METRO. Карта *1234. Доступно: 50000 RUB"

Паттерны:
- Сумма: (\d+\.?\d*)\s*RUB
- Место: RUB\.\s*(.+?)\.\s*Карта
- Карта: \*(\d{4})
```

### СБЕРБАНК
```
SMS от: 900
Пример SMS:
"Карта *5678. Покупка 3500 ₽. АШАН. Баланс: 75000 ₽"

Паттерны:
- Сумма: Покупка\s+(\d+)\s*₽
- Место: ₽\.\s*(.+?)\.\s*Баланс
- Карта: Карта\s+\*(\d{4})
```

## ЧТО ДЕЛАТЬ, ЕСЛИ НЕ РАБОТАЕТ?

### ПРОБЛЕМА 1: SMS не перехватываются
**Решение:**
1. Проверьте разрешения: Настройки → Команды → Разрешить автоматизацию
2. Убедитесь, что SMS приходит от правильного отправителя

### ПРОБЛЕМА 2: Паттерн не находит данные
**Решение:**
1. Скопируйте текст SMS
2. Проверьте на regex101.com
3. Скорректируйте паттерн

### ПРОБЛЕМА 3: HTTP-запрос не доходит до бота
**Решение:**
1. Проверьте URL (должен быть публичный домен, не localhost)
2. Убедитесь, что сервер запущен
3. Проверьте логи Flask
