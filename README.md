# 💰 Finance Bot — Telegram бот для учёта личных финансов

**Автоматический учёт транзакций из SMS и Email в Google Sheets**

---

## 🎯 ЧТО УМЕЕТ БОТ?

### ✅ Основные функции:
- ✍️ **Ручной ввод транзакций** через Telegram
- 📱 **Автоматический парсинг SMS** от банков (через iPhone Shortcuts)
- 📧 **Автоматический парсинг Email** от банков (через Gmail API)
- 📊 **Запись в Google Sheets** (автоматическая синхронизация)
- 📈 **Статистика** по категориям расходов
- 📋 **История транзакций** (просмотр последних операций)
- ✏️ **Редактирование и удаление** записей
- 🔔 **Уведомления админу** о новых транзакциях

## 🚀 БЫСТРЫЙ СТАРТ

### 1️⃣ Клонируйте репозиторий

```bash
git clone https://github.com/your-username/finance-bot.git
cd finance-bot
```

### 2️⃣ Установите зависимости

```bash
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3️⃣ Настройте .env файл

```bash
cp .env.example .env
nano .env  # Заполните все переменные
```

**Обязательные переменные:**
```env
TELEGRAM_TOKEN=your_bot_token_from_botfather
ADMIN_TELEGRAM_ID=your_telegram_id
ALLOWED_USER_IDS=123456789,987654321
SHEET_URL=https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit
WEBHOOK_URL=https://your-domain.com/webhook/transaction
```

### 4️⃣ Настройте Google Sheets API

1. Создайте проект в [Google Cloud Console](https://console.cloud.google.com/)
2. Включите Google Sheets API
3. Создайте Service Account
4. Скачайте JSON ключ → переименуйте в `finance-key.json`
5. Дайте доступ Service Account к вашей таблице

**Подробная инструкция:** [DEPLOYMENT.md](DEPLOYMENT.md)

### 5️⃣ Настройте Gmail API (для Email парсера)

1. Включите Gmail API в Google Cloud Console
2. Создайте OAuth 2.0 credentials
3. Скачайте JSON → переименуйте в `credentials.json`

**Подробная инструкция:** [DEPLOYMENT.md](DEPLOYMENT.md)

### 6️⃣ Запустите бота

```bash
python finance_bot.py
```

**Вы должны увидеть:**
```
INFO:root:🚀 Finance Bot запущен!
```

### 7️⃣ (Опционально) Запустите Gmail парсер

```bash
# В отдельном терминале
python gmail_parser.py
```

**При первом запуске:**
- Откроется браузер → войдите в Google → разрешите доступ
- Будет создан файл `token.json`

### 8️⃣ (Опционально) Настройте iPhone Shortcuts

**Инструкция:** [iPhone_Shortcuts_Setup.md](iPhone_Shortcuts_Setup.md)

## 🤖 КАК РАБОТАЕТ АВТОМАТИЗАЦИЯ?

### СЦЕНАРИЙ 1: SMS от банка (iPhone)

```
1. SMS приходит на iPhone
   ↓
2. iPhone Shortcuts перехватывает SMS
   ↓
3. Парсит: сумма, место, карта
   ↓
4. Отправляет HTTP-запрос в бота
   ↓
5. Бот спрашивает: "Хотите записать?"
   ↓
6. Вы выбираете категорию
   ↓
7. Бот записывает в Google Sheets
```

### СЦЕНАРИЙ 2: Email от банка

```
1. Email от банка приходит в Gmail
   ↓
2. gmail_parser.py проверяет почту каждые 30 сек
   ↓
3. Парсит: сумма, место, карта
   ↓
4. Отправляет уведомление в Telegram
   ↓
5. Вы выбираете категорию
   ↓
6. Бот записывает в Google Sheets
```

### СЦЕНАРИЙ 3: Ручной ввод

```
1. Открываете бота в Telegram
   ↓
2. Нажимаете "➕ Новая транзакция"
   ↓
3. Выбираете категорию, подкатегорию
   ↓
4. Вводите сумму, дату, комментарий
   ↓
5. Проверяете превью → "✅ Записать"
   ↓
6. Бот записывает в Google Sheets
```

## 🐛 TROUBLESHOOTING

### Бот не отвечает?
1. Проверьте токен в `.env`
2. Убедитесь, что бот запущен
3. Проверьте логи: `journalctl -u finance-bot -f`

### Ошибка доступа к Google Sheets?
1. Откройте `finance-key.json`
2. Скопируйте `client_email`
3. Дайте доступ этому email к таблице (Редактор)

### Gmail парсер не находит письма?
1. Проверьте, что Gmail API включен
2. Убедитесь, что письма не в спаме
3. Проверьте паттерны в `BANK_PATTERNS`

**Полный список решений:** [DEPLOYMENT.md](DEPLOYMENT.md)
